# Champion Heart in OpenClaw

## Install

1. Clone or copy this repository into your local OpenClaw skill or prompt workspace.
2. Read `.openclaw/INSTALL.md`.

This flow must stay consistent with `docs/install-model.md` and `README.md`.

## Use Pattern

1. Use `entry/SKILL.md` for gate selection.
2. Use `SKILL.md` for full champion behavior.
3. Use `commands/` for direct task entry.
4. Use `transmission/SKILL.md` if another loaded skill must be rewritten.

## Best Trigger Phrases

- `冠军之心`
- `冠军之心.skill`
- `冠军之心模式`
- `Champion Heart`
- `Champion Heart.skill`

These should be treated as strongest activation phrases.
