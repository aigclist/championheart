# Champion Heart in Codex

Codex does not require a plugin shell for this repository.

## Install

1. Clone this repository locally.
2. Read `.codex/INSTALL.md`.

This flow must stay consistent with `docs/install-model.md` and `README.md`.

## Use Pattern

1. Read `entry/SKILL.md` when routing matters.
2. Read `SKILL.md` when the session explicitly invokes `冠军之心` or `Champion Heart`.
3. Use `commands/` as manual task entrypoints.

## Best Trigger Phrases

- `冠军之心`
- `冠军之心.skill`
- `Champion Heart`
- `Champion Heart.skill`

These should be treated as highest-priority activation signals.
