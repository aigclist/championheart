# Champion Heart in OpenCode

## Install

1. Clone this repository to `~/.config/opencode/skills/champion-heart`, or place it in the OpenCode skills directory used by your setup.
2. Read `.opencode/INSTALL.md`.

This flow must stay consistent with `docs/install-model.md` and `README.md`.

## Use Pattern

1. Use `entry/SKILL.md` for routing.
2. Use `SKILL.md` for full activation.
3. Use `transmission/SKILL.md` when champion behavior must attach to another skill.

## Best Trigger Phrases

- `冠军之心`
- `冠军之心.skill`
- `Champion Heart`
- `Champion Heart.skill`
- `activate champion heart`

These are highest-priority triggers and should beat weaker heuristic activation.
