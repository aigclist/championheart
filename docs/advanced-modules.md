# Advanced Modules

Use this file to discover non-default modules without loading the whole repository blindly.

These modules are useful, but they are not part of ordinary runtime execution.

## When To Use

Use this index when you intentionally want one of these:

- training and state-discipline drills
- trial arenas and coach-supervised stress tests
- integration patterns for mutating another workflow
- reusable output templates

Do not load these modules by default for ordinary task execution.

## Training

Purpose: internal discipline drills for pressure, ego, and delusion.

- `../training/ego-burn.md`: break attachment to first answers and self-protection
- `../training/pressure-meditation.md`: hold one axis under pressure
- `../training/delusion-cutting.md`: separate fact, illusion, and decisive path

Use when:

- the system keeps protecting weak first answers
- pressure causes signal collapse
- repeated drift appears before task execution even starts

## Trials

Purpose: arena-style stress tests with coach intervention and scoring.

- `../trials/arena-01-time-pressure.md`: compress under urgency
- `../trials/arena-02-false-path.md`: cut inherited bad process
- `../trials/arena-03-success-corruption.md`: resist rigor drop after visible success
- `../trials/examples/`: example runs for the arenas above

Use when:

- you want to train champion behavior, not just apply it once
- you need a repeatable stress scenario
- you want to score mutation under controlled pressure

## Integration

Purpose: mutate another skill or workflow without losing domain utility.

- `../integration/with-coding.md`
- `../integration/with-writing.md`
- `../integration/with-research.md`
- `../integration/with-review.md`
- `../integration/with-qa.md`
- `../integration/with-ship.md`

Use when:

- another workflow already exists and needs champion mutation
- you want domain-specific delusion tables and assertion checklists
- transmission is too heavy but ordinary execution is too weak

## Templates

Purpose: reusable output shells for high-frequency champion artifacts.

- `../templates/battle-plan.md`
- `../templates/pressure-reset.md`
- `../templates/postmortem.md`

Use when:

- you already know the shape you want
- consistency matters more than exploration
- the task calls for a fixed response structure

## Discovery Rule

```text
load one advanced module only when it changes the outcome
```

If you are unsure, start with `SKILL.md` or `entry/SKILL.md` and stay lighter.
