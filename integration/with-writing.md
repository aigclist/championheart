# Integration: champion-heart + writing

Use champion-heart above writing skills when the work must produce consequence, not just pleasant language.

## Mutation

Champion-heart changes writing behavior:

- start from terminal effect on reader, not from surface polish
- cut decorative prose that does not move decision or understanding
- distinguish emotional force from rhetorical inflation
- end on verified clarity, not lyrical closure

## Sequence

1. define writing terminal state
2. define dominant reader constraint
3. identify main delusion:
   - sounding smart instead of moving truth
   - sounding moving instead of being precise
4. run writing skill
5. use coach if prose volume exceeds signal

## Rule

Good writing is not admired language.
Good writing changes the reader's state cleanly.
