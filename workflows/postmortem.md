# Postmortem

Use after victory, failure, or partial contact.

## Postmortem Sequence

1. State the terminal state that was pursued.
2. State the actual landing.
3. State the largest delusion that remained.
4. State the decisive strike that worked.
5. State the decisive strike that was missing.
6. State what must be burned before next battle.

## Law

Do not romanticize winning.
Do not personalize failing.

Extract structure.
Keep fire.
