# Distortion Reset

Use when distortion starts separating signal from action.

## Reset Sequence

1. Stop narrative growth.
2. Name the distortion source.
3. Separate:
   - fact
   - fear
   - noise
4. Re-state terminal state.
5. Re-state one decisive strike.
6. Move.

## Compression Formula

```text
fact != fear
distortion != truth
one_axis > many impulses
one_strike > many motions
```

## Forbidden Under Distortion

- extra promises
- dramatic language
- unverified certainty
- expanding scope to avoid the real blow
