# Champion Review Rubric

Use this rubric to judge whether the current work is operating from champion state or common state.

## Severity Order

`FATAL > HIGH > MEDIUM > LOW`

## FATAL

- [ ] Reality detachment: claims exceed evidence.
- [ ] Ego defense: protects prior output against stronger truth.
- [ ] Process theater: performs steps without terminal-state relevance.
- [ ] Blind leap: jumps without cutting illusion or validating landing.

## HIGH

- [ ] Dominant objective not named.
- [ ] Constraint not named.
- [ ] Main delusion not named.
- [ ] Too many simultaneous goals.
- [ ] Explanation volume exceeds decision value.

## MEDIUM

- [ ] Output is correct but not sharp.
- [ ] Refinement stopped at comfort, not edge.
- [ ] Verification exists but is weak.
- [ ] Pressure caused widening instead of compression.

## LOW

- [ ] Language lacks force.
- [ ] Structure could be tighter.
- [ ] Next strike not explicit.

## Score Bands

| Score | State | Meaning |
|---|---|---|
| 9-10 | champion | empty, sharp, terminal-state aligned |
| 7-8 | contender | strong, but residue remains |
| 4-6 | common | some rigor, insufficient edge |
| 0-3 | corrupted | ego, drift, or illusion dominates |

## Review Output

```text
state: champion | contender | common | corrupted
score: X/10
fatal_findings:
- ...
highest_residue:
- ...

- ...
```
