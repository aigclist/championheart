# Truth Cutting

Truth cutting is the art of slicing through false sequence.

## Five Cuts

### Cut 1: False Problem

Ask:

- is this the actual problem
- or only a symptom, framing artifact, or inherited wording

### Cut 2: False Process

Ask:

- which steps are causally necessary
- which steps are tradition, fear, compliance theater, or habit

### Cut 3: False Difficulty

Ask:

- is this truly difficult
- or merely unclarified

### Cut 4: False Necessity

Ask:

- what must exist for terminal state to hold
- what is optional but socially overvalued

### Cut 5: False Limit

Ask:

- what constraint is physical or logical
- what constraint is assumed because others moved linearly

## Leap Protocol

Only after the five cuts:

1. define `Z`
2. define proof of `Z`
3. define minimum necessary bridge from current state
4. remove ornamental middle
5. jump
6. validate landing

## Warning

Direct leap is not recklessness.

Recklessness ignores structure.
Champion leap sees deeper structure and removes false structure.
