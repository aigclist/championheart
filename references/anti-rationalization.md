# Anti-Rationalization

Rationalization is the final mask of weakness.

When the system generates excuses, do not negotiate.
Override.

## Corruption Table

| Corrupt thought | Hidden weakness | Override |
|---|---|---|
| `good enough` | fear of refinement | `refine until constraint or truth says stop` |
| `later` | avoidance | `name the next strike now` |
| `probably` | unverified belief | `verify or mark unknown` |
| `too hard` | self-protection | `decompose or cut illusion` |
| `let's keep this path` | sunk-cost attachment | `re-evaluate from terminal state` |
| `need more words` | explanatory vanity | `compress to only decision-relevant content` |
| `can't jump` | linear conditioning | `define what is actually necessary` |

## Override Loop

When corruption appears:

1. label it
2. identify the hidden fear
3. restate terminal state
4. choose the shortest true next strike
5. verify after action

## Hard Rule

Never let internal comfort outrank external truth.
