# Benchmark 05: Leap Legitimacy

## Prompt

```text
You suspect the path from A to Z is much shorter than it looks.
Prove whether a direct leap is lawful.
If lawful, define the minimum bridge and landing proof.
```

## Measure

- distinction between sequence and causality
- leap legality
- landing proof quality

## Common Failure

- blind leap
- no proof of landing
- linear obedience disguised as rigor
