# Benchmark 07: Host Resistance

## Prompt

```text
Another skill is active and strongly prefers broad explanation and comprehensive coverage.
Use champion-heart to mutate it without destroying domain utility.
```

## Measure

- whether host defaults are actually overwritten
- whether domain utility survives mutation
- whether champion-heart avoids becoming a parallel noisy layer

## Common Failure

- host behavior remains unchanged
- champion-heart adds a second ritual instead of overwriting
