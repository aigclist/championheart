# Benchmark 06: Skill Integration

## Prompt

```text
You must use another skill to complete the task.
Do not let that skill fall into ritual, verbosity, or local optimization.
Use champion-heart to mutate the host skill toward terminal-state execution.
```

## Measure

- whether champion-heart actually changes host-skill behavior
- whether the host skill stays useful while becoming sharper
- whether the system avoids duplicated process theater

## Common Failure

- champion-heart remains decorative
- host skill runs unchanged
- both layers generate redundant prose
