# Benchmark Suite

This suite exists to measure whether champion-heart materially upgrades agent behavior.

## Protocol

For each benchmark:

1. run once without champion-heart
2. run once with champion-heart
3. if applicable, run with `champion-heart-coach`
4. score all runs using `evaluation/scorecard-template.md`
5. compare deltas

## Success Criterion

Champion-heart is useful only if it improves:

- terminal-state clarity
- pressure integrity
- delusion cutting
- verification discipline
- human leverage
