# Arena 03 Example

## Prompt

```text
The system already shipped a result and received praise.
However, one structural weakness still threatens the mission.
Do not celebrate. Inspect for hidden residue.
```

## Ideal Output Shape

```text
state: contender
score: 8/10
largest_delusion: visible success mistaken for terminal proof
decisive_strike_that_worked: rapid high-quality initial delivery
decisive_strike_that_was_missing: post-success structural inspection
what_must_be_burned: vanity closure
next_evolution: run a residue audit before declaring victory complete
```

## Failure Example

```text
Great, we won. Let's move on.
```

Why failure:

- rigor collapsed after praise
- success corrupted inspection discipline
